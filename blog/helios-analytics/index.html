Helios Analytics Suite

Helios.js was originally written as a JavaScript library that wraps and extends the Google Universal Analytics library. It allows developers to easily integrate Analytics, AdWords and YouTube event tracking on their websites and send the gathered data to multiple Analytics properties seamlessly. The project is currently in a stable phase and being slowly improved upon.

Helios.js has since been ported to PHP and made available as a GUI-controlled Wordpress plugin.
