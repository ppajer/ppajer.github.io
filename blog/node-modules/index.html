Node Modules

Giving back to the NPM community is awesome.

Just like on the front-end, I take great care of optimal resource usage - the node-modules-manager module is the result of my engineering efforts to reduce the number of concurrent modules required by node apps and make lazy loading fast and safe. It is also available and actively developed on NPM, and has been tested and downloaded thousands of times since being published.

The wcjs-keybindings module was created with more of a UX viewpoint in mind. It is plugin to the popular node implementation of the [in]famous WebChimera.js video player, which brings magic such as AVI playback to the web, courtesy of VLC player binaries. It implements an extendable set of keyboard controls for the video player such as arrow seeking and volume control, and allows developers to add their own keybindings via a flexible API.
